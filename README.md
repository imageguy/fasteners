**Machine fasteners: STL library and generation code**

A large library of STL files describing metric and Imperial bolts, screws and nuts. There is also a Python package to generate custom fasteners.

See [Thingiverse **Machine bolt/screw/nut STL file library and generation
program**](https://www.thingiverse.com/thing:7337677) for the full
description.

For easy access to STL files, check stl/index.md and stl/index_raw.md.
They lead to pages for individual STL files. index.md leads to GitHub
object pages with preview and download button. index_raw.md lead to pages
that will just download the selected files.

There are also index.html pages that are useful if the whole tree is
copied to a file system.

gen_html.py and gen_md.py generate the files - a master file and files in
each of the leaves.
